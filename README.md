# ⛽ Petrol Price Dashboard

An interactive Power BI dashboard that provides a global overview of petrol prices across countries and regions. It highlights average prices, regional comparisons, price trends, and countries with the highest and lowest fuel prices.

### 🛠 Tools Used
- Power BI Desktop
- Microsoft Excel
- DAX
- Data Modeling

### 📌 Key Insights
- Highest petrol price: **Zimbabwe**
- Lowest petrol price: **Albania**
- Region-wise and country-wise price analysis
- Interactive filters for exploring price trends

### 📂 Files
- `Petrol Dashboard.pbix`
- `Petrol prices.xlsx`
- `petrol price.png`

Built to analyze global fuel price patterns through interactive and easy-to-understand visualizations.
